## 🎯 기획 의도

SNS의 발달로 전 세계는 빠르게 연결되었지만,  
그와 동시에 **대면 소통 감소**, **사회적 고립 증가**, **자기 비교 심화** 등의 문제가 나타났습니다.

IMAEZIM은 AR 기술을 통해:

- 현실 세계의 상호작용을 늘리고  
- 능동적인 참여형 SNS 경험을 제공하며  
- 사회적 단절을 완화  

하고자 하는 목표로 기획되었습니다.

## 🧭 기능 소개

### 🔹 AR SNS (핵심 기능)
- 원하는 위치(실내·실외·물건)에 **AR 메모** 남기기  
- 글 / 사진 / 영상 중 택 1  
- 근처 AR 메모 탐색  
- 다른 유저가 남긴 메모 감상

### 🔹 AR 길찾기
- SNS 게시물(AR 메모)까지 가는 AR 내비게이션 제공  
- GPS + AR 방향 안내

### 🔹 AR 게임
- AR 공간에서 즐길 수 있는 미니 게임  
- 유저 참여형 경험 제공

### 🔹 AR 퀴즈
- 위치 기반 AR 퀴즈  
- 퀴즈를 풀며 특정 위치로 이동하게 하는 체험형 기능
