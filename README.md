## 📘 프로젝트 소개

**IMAEZIM(이매짐)**은 AR 기술을 활용한 SNS 플랫폼으로,  
사용자가 **현실 세계에 글·사진·영상으로 된 AR 메모를 남기고**,  
다른 사람의 AR 게시물을 찾아가며 상호작용할 수 있는 새로운 형태의 소셜 경험을 제공합니다.

이매짐은 다음과 같은 AR 요소로 구성됩니다:

- 언제 어디서든 공유 가능한 **AR SNS 기능**
- 특정 SNS 게시물을 찾아가도록 돕는 **AR 길찾기**
- 재미 요소를 더한 **AR 게임**
- 위치 기반 **AR 퀴즈**

---

## 🎯 기획 의도

SNS의 발달로 전 세계는 빠르게 연결되었지만,  
그와 동시에 **대면 소통 감소**, **사회적 고립 증가**, **자기 비교 심화** 등의 문제가 나타났습니다.

IMAEZIM은 AR 기술을 통해:

- 현실 세계의 상호작용을 늘리고  
- 능동적인 참여형 SNS 경험을 제공하며  
- 사회적 단절을 완화  

하고자 하는 목표로 기획되었습니다.

## 🧭 기능 소개

### 🔹 AR SNS (핵심 기능)
- 원하는 위치(실내·실외·물건)에 **AR 메모** 남기기  
- 글 / 사진 / 영상 중 택 1  
- 근처 AR 메모 탐색  
- 다른 유저가 남긴 메모 감상

![IMAEZIM_optimized_smaller](https://github.com/user-attachments/assets/717ee5a7-747e-471f-b773-73ec2f531cf4)





### 🔹 AR 길찾기
- SNS 게시물(AR 메모)까지 가는 AR 내비게이션 제공  
- GPS + AR 방향 안내

### 🔹 AR 게임
- AR 공간에서 즐길 수 있는 미니 게임  
- 유저 참여형 경험 제공

### 🔹 AR 퀴즈
- 위치 기반 AR 퀴즈  
- 퀴즈를 풀며 특정 위치로 이동하게 하는 체험형 기능
